# 2D Tetris - Enhanced

A 2D Tetris game with customizable difficulty settings and a lives system.

## Features
- **Difficulty Levels**: Choose between Easy, Difficult, or Extreme modes.
- **Lives System**: Track and manage your lives to survive longer.
- **Dynamic UI**: Real-time updates for lives and game state.

## How to Play
1. Select your preferred difficulty level.
2. Click "Start Game" to begin.
3. Use the arrow keys to move and rotate the pieces.
4. Clear lines to score points and avoid stacking to the top.
5. Manage your lives to extend your gameplay.

## Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/Mavasahikona/2D-Tetris-Enhanced.git
   ```
2. Open `index.html` in your browser.
